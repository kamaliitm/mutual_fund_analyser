Mutual Funds - Evaluation by Rolling Returns & Risk


Objective
- Our goal is to evaluate various mutual funds, compare by categories such as equity, debt, hybrid (equity oriented) by checking the rolling returns and standard deviations data for however long we can go back in time. Maybe 20 years. We should be able to visualize the rolling returns and standard deviation metrics in a spreadsheet like a google spreadsheet or a format that is compatible with WPS.


What needs to be done?
- Consider a configurable list of mutual funds - example: ICICI prudential eq & debt fund, Parag Parikh flexi cap fund et al.
- For each fund, from the public APIs, get the daily prices for however long back data we can go - say 15 years.
- We should be able to write the daily prices data to a csv file (or a format that can work in google spreadsheet as well as WPS)
- We should calculate the rolling returns for 3 years, 5 years, 7 years, 10 years (if data is available) and write to the file
- We should be able to calculate standard deviations to measure the risk for the above same periods and write to the file.
- Compare the mutual funds against rolling returns and standard deviations with the normalized data.
